
# Enunciado

Escribir la clase `Sublista` para que contenga un método de clase `hacerSubLista`, de
manera que dicho método devuelva como resultado una nueva lista con todos los elementos
de otra lista dada como parámetro entre 2 índices también dados como parámetros. 

Las listas indicadas son de números enteros (`Integer`) y para utilizarlas se supone
que se puede utilizar la interfaz `IList`, es decir, `IList<Integer>`. La `ILista` es 
la que está definida en tads1920.jar. 

Indicar en forma de comentario de javadoc las precondiciones y/o postcondiciones
con las que se ha construido la solución.

Comprobar la solución propuesta en un main dentro de la propia clase.
