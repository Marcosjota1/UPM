
# Trabajar con clases y objetos

## Objetivo

Se trata de escribir distintas clases para describir los fundamentos más básicos
de la POO con Java.

## Atributos y métodos

Escribir una clase Punto que tenga las siguientes características:

- Tenga dos atributos para guardar sus coordenadas x, y.
- Tenga un método para que se pueda convertir a String y escribir en la salida estándar.

Escribir un clase Principal para probar la clase punto. Implica:

- Crear un objeto Punto.
- Escribir ese objeto en la salida 

## Constructor

Completar la clase para que:

- Se pueda dar valor inicial a las coordenadas.

## Métodos que modifican el estado

Añadir un método que modifique el punto modificando su posición mediante
dos incrementos, uno para cada coordenada.

## Trabajar con varios objetos

Sobre el programa principal (clase Principal), añadir algunas sentencias extra:

- Crear otro objeto Punto
- Distinguir entre objetos Punto y referencias a objetos Punto
- Usar variables de tipo double, ¿Qué diferencia hay entre objetos y primitivas?
