# Proyecto GUI

Este proyecto contiene clases que usan las librerías de Java 
para interfaces gráficas (las del propio JDK): awt y swing.

La idea es usarlas como ejemplo de relaciones entre objetos,
también llamadas composición. 

## Conceptos a revisar

1. Comprobar dónde se crean los objetos
1. Entender que cuando se crean los objetos de una clase inmediatamente se crean objetos de otras
1. Entender la sintaxis de inicialización de atributos mediante constructor y mediante inicialización directa
1. Ver cómo se consigue que un objeto se relacione con otro
1. En un array estudiar las diferencias entre array de primitivas y array de objetos
