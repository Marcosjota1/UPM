
# Instruciones para bajar la librería

## Explicación

La carpeta tads de este proyecto es un submodulo de Git.

Es decir, la carpeta tads apunta a un otro repositorio
que hay que bajar de forma explícita. No basta con hacer
un pull.

## Comandos

Hay que abrir una terminal de git en cualquier carpeta del 
repositorio (git bash en Windows), una terminal cualquier
en GNU-Linux.

En la terminal ejecutar:

```
git submodule init
git submodule update
```
