
# Niños y canicas

Vamos a escribir un programa que lea desde teclado los nombres de
algunos niños y el número de canicas que tiene cada uno.

Luego vamos a buscar:

1) El niño que tiene más canicas
2) El niño que tiene menos canicas

Lo vamos a hacer de dos maneras:

## Primera opción:

Tenemos la siguiente *restricción*: NO podemos hacer ninguna clase nueva.

## Segunda opción:

Podemos hacer una clase nueva que tenga como atributo el niño y el número de canicas.
